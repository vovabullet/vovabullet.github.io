Autoservice CRM – CRM-система для автосервиса

↗ github
Java 21Spring BootSpring SecurityPostgreSQLDockerELKGatling

Коммерческая разработка по заказу клиента, выполненная самостоятельно от архитектуры до деплоя (легла в основу дипломной работы). CRM для автосервиса: клиенты и автомобили, заказы с жизненным циклом статусов, услуги, рабочие задачи механиков и акты осмотра с фотофиксацией.

  • Спроектировал REST API на Spring Boot с чистой слоистой архитектурой (controller / service / repository / DTO / mapper).
  • JWT-аутентификация на Spring Security с ролевой моделью доступа; доменные жизненные циклы заявок и задач.
  • PostgreSQL (JPA/Hibernate), документация API через OpenAPI/Swagger, глобальная обработка ошибок и валидация.
  • Контейнеризация в Docker Compose, централизованные логи через ELK, нагрузочное тестирование Gatling.
  • Проектная документация: UML/BPMN-диаграммы; веб-клиент на HTML/CSS/JavaScript поверх API.

Hard-Loot – 2D action-roguelike

↗ github
GodotGDScriptTilemap

Top-down 2D action-roguelike с перманентной смертью и рандомным лутом: короткие напряжённые вылазки в опасные зоны ради добычи.

  • Боевая база на стамине: бег, атака, блок, отталкивание и фидбек ударов.
  • Data-driven лут (префиксы, суффиксы, редкости) на JSON-таблицах; инвентарь-сетка с drag&drop.
  • Враги на конечных автоматах (State Machine: патруль → агро → атака → отступ); фиксированная карта с рандомным наполнением.
